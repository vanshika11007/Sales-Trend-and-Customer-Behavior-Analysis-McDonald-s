# Sales-Trend-and-Customer-Behavior-Analysis-McDonald-s
Data analysis and interactive dashboard project on McDonald's sales data using Excel and Power Query to uncover revenue trends, customer behavior, and product performance insights.
# McDonald's Sales Analysis

## Overview

This project analyzes McDonald's transactional sales data to identify revenue trends, customer purchasing behavior, and product performance patterns. The goal is to build an interactive Excel dashboard that converts raw sales data into meaningful business insights.

The analysis focuses on category-wise performance, time-based sales distribution, and item-level contribution to overall revenue.

---

## Dataset Description

The dataset consists of 2 relational tables:

- MacD Items  
- Order Details  

### MacD Items Table
- Item Name  
- Category  
- Price  

### Order Details Table
- Order ID  
- Month Name  
- Day Name  
- Hour  
- Time Slot (Morning, Afternoon, Evening, Night)  
- Weekday/Weekend  
- Sales  

Both tables are connected using **Item Name** to enable category-level and item-level analysis.

Time Period: Static transactional dataset

---

## Objectives

- Analyze overall sales performance  
- Identify top-performing categories and items  
- Study monthly and time-based trends  
- Compare weekday vs weekend sales  
- Identify peak business hours  
- Build an interactive dashboard  

---

## Dashboard Features

- Total Revenue KPI  
- Total Orders KPI  
- Average Order Metrics  
- Revenue by Category  
- Monthly Sales Trend  
- Sales by Time Slot  
- Weekday vs Weekend Comparison  
- Top 5 Menu Items
